    Hydralette

    Create complex configurations in a simple, pythonic way!

    Installation

    pip install hydralette
# OR
poetry add hydralette

    Changelog

    • v0.1.3 at commit 90655ca
      • Add automatic generation of configs from signatures via from_signature and config_from_signature
    • v0.1.2 at commit 5848f43
      • Add support for config groups, references, validation and type conversion
      • Add CLI help pages

    Features

    • [x] Build configs like dataclasses | Example
    • [x] Automatically generate configs from class/function signatures | Example
    • [x] Effortless CLI from Config classes | Example
    • [x] Config groups to swap whole components | Example
    • [x] Referencing other config values to reduce redundancy | Example
    • [x] Type Conversion from CLI | Example
    • [x] Value Validation | Example
